## Dashboard/Reports Center

### Compliance Report

The Compliance Report tracks and manages compliance requirements based on orders

generated from a deal for an employer organization (EO) and sub-employer organizations.

![](./employer-portal-dashboard.assets/6a68a8e18a51efa899fd8b9f2d47043f.png)

### Quarterly Summary By Body Part

The Quarterly Summary by Body Part Report contains specific fields (e.g., Body Part Affected, Totals by Quarter, Totals by Year) grouped by client and location to track and manage injury-related data.

![](./employer-portal-dashboard.assets/14a0f24a7519ee823dc05ef55c3261ff.png)

### Visit Detail By Status Report

The Visit Detail by Status Report contains specific fields (e.g., EO, Sub EO, Location Where Services Were Rendered, Service Date, Employee name, Encounter Type, Procedures and Orders Status) to track and manage visit-related data.

![](./employer-portal-dashboard.assets/8e6f5ee72a4e4fbb9f911053647ead14.png)

### Visit Log Report

The Visit Log report exports encounter data specific to an employer organization. Information includes the employee name, clinic location where services were performed, MR number, Employee name, Position Title, and Service date. Add'l columns also display Health Surveillance clearance and Status if the encounter listed is related to a surveillance visit.

![](./employer-portal-dashboard.assets/1a13adf21fa22b874bc9fbcae536fb8d.png)

### Visit Status By Service Type Report

The Visit Status by Service Type Report contains specific fields (e.g., EO, Sub EO, Location Where Services Were Rendered, Service Date,Employee ID and Name, Encounter Type and Procedures) that allows the supervisor to view a detailed status of what the employee's clearance is pending for.

![](./employer-portal-dashboard.assets/084e7aebb3c468f91b32b217d6f3a0af.png)
