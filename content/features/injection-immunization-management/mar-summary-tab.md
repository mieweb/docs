The MAR Summary tab is a view only chart tab that displays a one page summary of the patient's medication administration history, immunization history, allergy injection vial history and medication history. The summary displays in a DataVis format, so users may easily manipulate the output display and/or create custom perspectives. By default the Medication Administration, Immunizations and Vials display all entered injections. The Medication History DataVis defaults to active medications only. Users may use the perspective drop down on the DataVis to select other preconfigured views.
