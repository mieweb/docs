## **Sample Bash Script**  


```
  
#!/bin/bash  
  
if [ $# -lt 5 ]; then  
	echo Usage: $0 interface message user password url  
	exit 1  
fi  
  
INTERFACE=$1  
MESSAGE=$2  
LOGINUSER=$3  
LOGINPASS=$4  
URL=$5  
  
curl -i \  
-F "f=wchl7" \  
-F "interface=$INTERFACE" \  
-F "login_user=$LOGINUSER" \  
-F "login_passwd=$LOGINPASS" \  
-F "message=@$MESSAGE" \  
"$URL"  

```
