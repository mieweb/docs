**Subcategory Name**: Expanded Practice Access  
**Description**:Â User participation in the Quality Payment Program website testing is an activity for eligible clinicians who have worked with CMS to provided substantive, timely, and responsive input to improve the CMS Quality Payment Program website through product user-testing that enhances system and program accessibility, readability and responsiveness as well as providing feedback for developing tools and guidance thereby allowing for a more user-friendly and accessible clinician and practice Quality Payment Program website experience.




Supporting Documentation  
Documentation of input to improve the CMS Quality Payment Program website through product user-testing aimed at enhancing system and program accessibility, readability and responsiveness and Provide feedback for developing tools and guidance for a more efficient and accessible clinician and practice QPP website experience.
